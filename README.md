# ROCHAO_KeyGenerator

Instalation directions: <br>
1. Git Clone: `git clone https://github.com/ROCHAO-Repos/ROCHAO_KeyGenerator.git`
1. Install Curl via vcpkg : https://curl.se/docs/install.html
1. Install INI Not Invented Here via vcpkg: https://github.com/benhoyt/inih
1. Open solution and build in VS 2022 Community Edition
